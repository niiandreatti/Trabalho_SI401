Seja bem-vindo(a) a uma emocionante jornada no mundo da programação web através da disciplina SI401. Neste espaço, vamos explorar os principais conceitos e tópicos abordados nesse curso empolgante que mergulha fundo no universo da criação de sites e aplicações web interativas.



O que é a disciplina SI401?

A disciplina SI401 é um curso dedicado ao estudo da programação web. Ao longo do curso, você terá a oportunidade de aprender sobre as tecnologias, linguagens e frameworks que sustentam a construção de websites modernos e envolventes. Desde a estruturação básica de uma página até a implementação de recursos avançados, a SI401 oferece uma base sólida para aqueles que desejam dominar a arte da programação web.



O Que Esperar?

Ao iniciar sua jornada de aprendizado na disciplina SI401, você encontrará uma variedade de tópicos essenciais no campo da programação web. Abaixo estão os principais temas deste curso:



Nosso Caminho

HTML

HTML é uma linguagem de marcação essencial para construir páginas da web, usando tags para estruturar elementos como textos, imagens e links. Ele define a base da apresentação e interação na internet

CSS

CSS, é crucial na web ao permitir que desenvolvedores definam a aparência visual de páginas HTML, incluindo cores, fontes e layouts, garantindo uma experiência estilizada e coesa para os visitantes.

JS

JavaScript é uma linguagem de programação amplamente utilizada na web, capacitando interatividade dinâmica ao permitir que os desenvolvedores criem e controlem elementos como animações, e etc.
